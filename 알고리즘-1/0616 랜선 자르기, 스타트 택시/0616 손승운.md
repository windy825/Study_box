#### 랜선 자르기(백준)

```python
#시간초과 나왔습니다....(10%)

import sys
a, b = map(int, input().split())      #입력받기
arr = [int(sys.stdin.readline()) for _ in range(a)]


avr = sum(arr) // b
c = avr

while c:
    d = 0
    for i in range(len(arr)):
        d += arr[i] // c
    if d == b:
        break
    elif d > b:
        c += 1
        continue
    else:
        c -= 1
        continue
print(c)
```
