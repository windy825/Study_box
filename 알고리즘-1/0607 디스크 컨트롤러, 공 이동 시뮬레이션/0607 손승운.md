#### 디스크 컨트롤러

```python
def solution(jobs):
    jobs.sort(key=lambda x : (x[0] + x[1]))
    f = 0
    times = []
    for i in range(len(jobs)):
        t = f + jobs[i][1] - jobs[i][0]
        times.append(t)
        f += jobs[i][1]
    answer = sum(times)
    return answer/len(jobs)
```
