#### 가장 큰 수

```python
죄송합니다 찾아봤습니다
def solution(numbers):
    numbers = list(map(str, numbers))
    numbers.sort(key = lambda x: x*3, reverse=True)
    return str(int(''.join(numbers)))



다른 풀이(원래 제가 풀고자 한 풀이)

쟁점
1. 이 문제는 정렬문제이다.
2. 입력 값의 자릿수가 다르다.

쟁점을 고려한 문제풀이
1. 한자리 숫자*100, 두자리 숫자*10, 3자리 숫자를 정렬한다.
2. 정렬한 숫자 중 최고자리 숫자가 같으면 비교.(아래는 비교예시)
	2-1. if 원래 한자리 숫자를 제외한 숫자가 최고자리 숫자가 최고자리 다음 숫자보다 작으면 후순위 
    2-2. elif 최고자리 숫자가 최고자리 다음 숫자보다 크면 선순위
    2-3. elif 둘째 자리 숫자까지 같으면 셋째 자리도 위의 과정을 수행
(ex : [6, 10, 2]일때, [600, 100, 200]으로 바꾼 후 정렬하면 [100, 200, 600]이다. 이를 pop한 후 결과 리스트에 append하면 6210으로 할 수 있다.)
(ex : [36, 3]이라면 [300, 360]인데, 360의 경우를 보면 3<6이므로 선순위로 등록. 따라서 363이다)
(ex ; [52, 5]이라면 [500, 520]인데, 520의 경우를 보면 5>2이므로 후순위로 등록, 따라서 552가 된다.)
(ex : [738, 78]이라면 [738, 780]인데, 738은 7>3이므로 후순위이고 78은 7>8이므로 선순위이다. 따라서 78738이다)
(ex : [46, 465]이라면 [460, 465]인데, 두 원소 모두 같으니 셋쨰 자리도 비교. 따라서 46546이 된다.)
```
